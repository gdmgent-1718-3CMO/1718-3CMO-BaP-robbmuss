Drukwerk

Kom meer te weten over de verschillende zaken die gedrukt worden.

Briefhoofd + envelop

Formaat: Briefhoofd: 210mm x 297mm / Amerikaanse Envelop: 114mm x 229mm Kleur: Recto 2 kleuren
Corps: Briefhoofd: Lopende tekst = 12pt Light / Handtekening = 12pt Semibold / Titel = 15pt Semibold - Envelop: Adres = 11pt Regular / Naam = 15pt Semibold

Belangrijk!

Het aanpassen van de header (logo + gegevens) en de footer (gegevens + illustratie) is niet toegelaten. Datum, inhoud en handtekening is wel toegelaten.



Businesscards

Formaat: 55 x 85mm Kleur: Recto / Verso 2 kleuren Corps: Naam = 30pt Bold / Informatie = 8pt Light en Regular

Telex

Formaat: 210mm x 297mm Kleur: Enkele pagina = Recto Quadri - Meerdere pagina's = Recto / Verso Quadri
Afbeeldingen: grijze vlakken = op te vullen met afbeeldingen.
Corps: Lopende tekst = 11pt Light / Titel = 30pt Semibold / Jaargang = 15pt Light / Subtitel = 15pt Light

Belangrijk!

Het aanpassen van de header (logo + gegevens) is niet toegelaten. Header afbeelding is vrij te kiezen. Gebruik van icons moet gebeuren op de aangewezen manier, maar het gebruik is niet verplicht. Foto's op andere plekken in de tekst worden geplaatst zoals de header. Afbeeldingen van evenement posters zijn te plaatsen zoals in de template.

Folder

Formaat: 210mm x 297mm Kleur: Recto / Verso Quadri Pagina's: Cover + TOC + Contact + Back = (6p) + X zelf in te vullen pagina's.
Corps: Lopende tekst = 11pt Light / Subtitels = 15pt Semibold / TOC = 20pt Light Semibold / Onderwerp cover = 30pt Semibold
Afbeeldingen: grijze vlakken = op te vullen met afbeeldingen.

Belangrijk!

In de template zijn verschillende voorstellen te vinden voor verschillende pagina's. Pagina's met afbeeldingen, zonder afbeeldingen...
Deze mogen in verschillende volgorde gebruikt worden. Op elke pagina staat er bovenaan "Onderwerp" hier kan je het onderwerp invullen waar het op deze pagina over gaat. Pagina te leeg en geen tekst meer om op te vullen? = Afbeeldingen invoegen op manier zoals andere afbeeldingen (afbeelding breedte = tekst kolom breedte).

Formaat: 150mm x 70mm Kleur: Recto Quadri
